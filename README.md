# Workforce Productivity & Capacity Analysis  

This project analyzes how employee productivity, capacity, satisfaction, and manager influence intersect inside a workforce.  
Using a synthetic dataset of 200 employees, I explored key operational and people-analytics metrics that help organizations detect burnout early, optimize workload distribution, and improve manager effectiveness.
